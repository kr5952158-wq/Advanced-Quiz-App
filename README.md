# Advanced Web Development Quiz App

An interactive and responsive Quiz Application built using **HTML, CSS, JavaScript, and Bootstrap**.  
This project tests knowledge of Web Development concepts including HTML, CSS, and JavaScript.

## Features

- 20 Multiple Choice Questions  
-  5-Minute Countdown Timer  
-  Previous & Next Navigation  
-  Real-Time Question Counter  
-  Answer Selection & State Management  
-  Automatic Score Calculation  
-  Highlight Correct & Wrong Answers  
-  Show/Hide Unanswered Questions  
-  Back to Start / Restart Option  
-  Fully Responsive Design  
-  Modern Gradient UI  

---

##  Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Bootstrap 4

##  Concepts Implemented

- DOM Manipulation  
- Event Handling  
- Timer Functions (setInterval)  
- Dynamic Content Rendering  
- Conditional Logic

- Responsive UI Design

## How to Run the Project

1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Click on **Start Quiz**  
4. Attempt the questions and submit
